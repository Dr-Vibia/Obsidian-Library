13.同余
1. $x \mod a$ : 这个命令将在 x 和 mod 之间产生很大的间隔.
2. $x \bmod a$ : 这个命令在 x 和 mod 之间产生的间隔比较合适.
3. $x \pmod a$ : 这个命令将在上面的基础上添加小括号.
4. $x \pod a$ : 省略 mod .
